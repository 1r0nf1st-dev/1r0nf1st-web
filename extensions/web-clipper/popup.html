1r0nf1st Notes Clipper

What it does

Save any webpage to 1r0nf1st Notes in one click. (This is the 1r0nf1st extension—not Evernote or others.)

Setup (in order)
  1. Log in at your site
  2. Go to Notes → open Tools sidebar → Web Clipper
  3. Generate token and copy it (starts with nc_)
  4. Paste below, enter Site URL, then Save
